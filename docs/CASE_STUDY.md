# Case Study: BeeHive Dashboard

## Resumen

Dashboard de productividad gamificado con Pomodoro, Kanban, XP quests y autenticación. Diseñado para ayudar a usuarios a gestionar tareas con motivación lúdica.

## Problema

Las herramientas de productividad tradicionales son aburridas y no logran mantener el engagement del usuario.

## Solución

Combinación de técnicas de productividad (Pomodoro, Kanban) con gamificación (XP, niveles, quests) y datos visuales (estadísticas SVG).

## Stack

Next.js 15 + React 19 + TypeScript + Tailwind CSS 4 + Neon/PostgreSQL + Drizzle ORM + Dexie.js + @dnd-kit
